README

A program to add search terms from terminal and open browser with specified amount of the top search results.

The script prompts 
1. Search term. If left empty the script will grab the search term from clipboard
2. How many pages you want the script to open. If left empty the program will open 3 tabs by default.

REQUIRED PACKETS
Needed packages:
	googlesearch-python
	pyperclip

Install it with pip install googlesearch-python

------
Idea from Al Sweigart's Automate the boring stuff. Google has changed their code since (printed in 2015) so the script specified in the book no longer works.
